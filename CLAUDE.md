# hafiy.my

Minimalist personal website inspired by Derek Sivers.
Hand-coded HTML, no frameworks, no CSS libraries.
Hosted on GitHub Pages with custom domain via Exabytes DNS.

## Design philosophy
- Brutally simple, content-first
- No JavaScript unless absolutely necessary
- Clean, readable HTML

## Structure
- index.html — homepage
- about.html — about page
- CNAME — custom domain config (hafiy.my)

## Hosting
- GitHub Pages user site (nazirulhafiy.github.io)
- Custom domain: hafiy.my
- DNS: A records + CNAME configured via Exabytes
